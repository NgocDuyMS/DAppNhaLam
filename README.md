# DApp Voting - NhaLam

Ứng dụng bầu cử ẩn danh sử dụng Blockchain (Oasis Sapphire).

## Tech Stack
- Frontend: Next.js, Tailwind
- Backend: Solidity, Hardhat
- Blockchain: Oasis Sapphire

## Project Structure

- frontend-web: giao diện người dùng
- backend-contracts: smart contract

## Setup

### Backend
cd backend-contracts
npm install
npx hardhat compile

### Frontend
cd frontend-web
npm install
npm run dev

## Features
- Tạo cuộc bầu cử
- Thêm ứng viên
- Bỏ phiếu ẩn danh
- Hiển thị kết quả
